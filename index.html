☁️ GitHub Sync

Your library syncs via a private GitHub Gist — free, no extra account needed.
Setup (one time per device):
1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Click Generate new token (classic)
3. Check only the gist scope
4. Copy the token and paste below
✅ Token saved on this device
